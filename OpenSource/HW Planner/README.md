# Homework Planner

A simple Python application to manage homework subjects.

## Usage

Run the application:

```bash
python main.py
```

This will start the subject selector where you can:
- Choose to add or create subjects
- View existing subjects

## Features

- Create and manage subjects for homework
- Interactive subject selection

## Troubleshooting

- Ensure Python 3 is installed
- Run from the HW Planner directory
